# Zombie Survival
A top-down 2D zombie shooter built in Unity/C#.

## Play it
https://jsnaggy.itch.io/zombie-survival

## What I built
- Wave-based enemy spawner with escalating difficulty
- Mouse-aimed shooting system with bullet physics
- Enemy AI with chase behavior
- Dynamic HUD with health bar and wave counter

## Built with
Unity · C# · Cinemachine · TextMeshPro
